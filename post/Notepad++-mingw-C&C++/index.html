Notepad++ 配置MinGW编译C/C++

2020-08-09

一、安装Notepad++

自行百度，这里不再赘述。

二、安装MinGW

1.MinGW安装器

https://osdn.net/projects/mingw/releases/

下载后打开安装，勾选需要的包，然后在Installation中点Apply Changes

2.MinGW-W64离线包

打开https://sourceforge.net/projects/mingw-w64/files/，下载x86_64-win32-seh 后解压到合适位置即可

三、配置系统变量

无论是使用安装器安装还是使用离线包，都必须配置系统变量之后才能使用

在“此电脑”右键-属性，打开高级系统设置-环境变量，在系统变量下找到Path,点进去进行编辑，将MinGW文件夹中的bin文件夹路径添加进去

四、配置notepad++

1.点击运行-运行，或按F5，输入cmd /k gcc -o "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" "$(FULL_CURRENT_PATH)" && CLS && "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" & PAUSE & EXIT点击保存并定义为合适的名称

2.写一段完整的C语言程序

#include<stdio.h>
int main()
{	
	printf("hello world\n");
	return 0;	
}

3.点运行，选择刚保存的命令

运行成功！
4.如果是C++，则在运行内输入cmd /k g++ -o "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" "$(FULL_CURRENT_PATH)" && CLS && "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" & PAUSE & EXIT

五、补充

  • 编译：
    cmd /k gcc -Wall -o "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" "$(FULL_CURRENT_PATH)" & PAUSE & EXIT
  • 运行：
    cmd /k "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" & PAUSE & EXIT

关于脚本行：cmd /k gcc -o "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" "$(FULL_CURRENT_PATH)" && CLS && "$(CURRENT_DIRECTORY)\$(NAME_PART).exe" & PAUSE & EXIT

  • cmd /k是弹出cmd窗口并执行后续指令。&和&&表示连接作用，说明有多行指令合为一行。

  • gcc -o “$(CURRENT_DIRECTORY)$(NAME_PART).exe” “$(FULL_CURRENT_PATH)”编译生成.exe文件，输出源文件所在目录。注意这里需要有引号，目的是为了在目录及文件名存在空格的情况下也能够正常运行。

  • && CLS 清屏。&&的意义在于，前面的语句出现错误，将不会执行该句。

  • && “$(CURRENT_DIRECTORY)$(NAME_PART).exe” 运行刚刚生成的.exe文件。如果先前的编译错误，将不会运行。

  • & PAUSE 暂停，提示”按任意键继续“。这里无论前面有否错误，都将执行。因此我们看到的是编译错误，显示错误信息并暂停的窗口。

  • & EXIT 退出。
