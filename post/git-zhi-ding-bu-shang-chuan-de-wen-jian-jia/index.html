Git 指定不上传的文件夹

2020-05-14 1 min read # Git

1.文件夹内右键git bash，输 touch .gitignore，注意中间有空格。

2.编辑器打开生成的 .gitignore 文件，加入:

node_modules

/*以及其他你想要忽略的文件或文件夹*/

3.保存。

4.以后再add到暂存区的时候就会忽略你配置的文件或文件夹了。

一般来说：git是的默认项目里也是有配置的：

忽略具体的文件或文件夹：

/target/  // 忽略这个target 目录
log/*     // 忽略log下的所有文件
css/*.css // 忽略css目录下的.css文件
